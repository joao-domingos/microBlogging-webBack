## **Project:** Micro Blogging - web back end

Projeto para aula de programação web back-end com foco em banco de dados e servidor, logo sem a parte web. Esse projeto cria um sistema que permite ao usuario criar posts e depois busca-los, usando node.js e mondodb, sem o auxilio de express.

## Como foi feito:

**Tech used:** express, node, mongodb, mvc, git

## Como rodar no linux: 

**1** - make sure u have nodejs and mongodb installed
**2** - inicialize your mongodb service
**3** - start app using $ node index.js

## O que eu aprendi:

Primeiramente, esse projeto foi o meu primeiro contato com um backend real. Anteriormente, tive a experiência de aprender mais sobre APIs, tratamento de exceções, mas nunca tendo que conectar bando de dados com o servidor e menos ainda tendo que trabalhar com roteamento

Tive bastante dificuldade de entender o funcionamento da conexão com o banco de dados, principalmente sobre como lidar com o banco dentro das funções da API.
